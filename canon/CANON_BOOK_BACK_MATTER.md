# Canon Book Back Matter

This document defines the canonical back matter and spine architecture
for all books in *The Wellings* franchise.

This architecture applies to all sets, all books, and all editions.

---

## Scope

- Applies after story content concludes
- Governs interior pagination end rules
- Defines exterior back cover and spine contents
- Ensures series continuity and catalog clarity

---

## Back Matter Sequence

### 1. Conditional Blank Page

- Insert a blank page only if required
- Rule:
  - If final story page is odd-numbered (recto): no blank page needed
  - If final story page is even-numbered (verso): insert one blank page

---

### 2. Full Series Page (Recto)

- Displays:
  - All sets
  - All book titles
- No story content

---

### 3. Blank Page (Verso)

- Back of full series page

---

### 4. Back Cover (Exterior)

- Illustration:
  - Same character as Dedication Poem Page
  - Chasing the same butterfly
  - Different pose and/or position
- Exterior only
- **Not counted as interior pagination**

---

## Spine (Print Editions Only)

- Vertical: Author last name
- Separation space
- Vertical: Full book title
- Separation space
- Horizontal: Series number / Book number indicator
- Separation space
- Publisher logo
- Publisher name
- Exterior only
- **Not interior pagination**
