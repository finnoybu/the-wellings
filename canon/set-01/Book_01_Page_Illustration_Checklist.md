# Book 1 — Page-Level Illustration Checklist
## The Wellings and the Very Busy Day

This document defines the required visual elements for each illustrated page
in Book 1. It does not prescribe composition, emotion, or style.

All illustrations must comply with existing canon.

**Environment:**  
- Welling House — Front Yard

**Characters Present:**  
- Arthur  
- Eleanor  
- Oliver  
- Miso

**Required Visual Elements:**  
- Family present together outdoors  
- House visible in background  

**Explicit Exclusions:**  
- Interior spaces  
- Porch or doorway emphasis
