---

FRANCHISE & SET NAMES

Franchise Title:
- The Wellings

The franchise consists of four thematic sets.
Each set defines emotional scope and thematic focus only.
Sets do not imply plot outcomes, character arcs, or resolutions beyond tone.

Set names are canonical and must be used consistently.

---

SET ONE (Books 1–6)
Title: The Days We Come Home

Franchise Usage:
- The Wellings — The Days We Come Home

Thematic Focus:
- Return
- Familiarity
- Daily rhythm
- Warmth
- Observation

Intent:
- Anchors the series in the feeling of coming back
- Establishes safety, routine, and emotional grounding
- Stories feel remembered rather than narrated
- Emphasis is on presence, not change

Set One stories may include:
- Busy days
- Waiting
- Small questions
- Quiet warmth

Set One must NOT:
- Resolve long-term themes
- Introduce future-set ideas
- Frame moments as lessons or conclusions

---

SET TWO (Books 7–12)
Title: The Time Things Take

Franchise Usage:
- The Wellings — The Time Things Take

Thematic Focus:
- Time
- Patience
- Waiting
- Settling
- Gradual understanding

Intent:
- Serves as a thematic mirror to Set One
- Shifts focus from place (home) to time (duration)
- Changes occur slowly and without announcement
- Tone becomes slightly more reflective, but remains gentle

Set Two stories may include:
- Lost or unnoticed time
- Waiting without urgency
- Quiet acceptance
- Emotional settling

Relationship to Set One:
- Place → then Time
- Return → then Rest

Together, Set One and Set Two form a paired foundation for the series.

---

SET THREE (Books 13–18)
Title: The Changes We Notice

Status:
- Reserved for later canon

Thematic Intent (High-Level Only):
- Observation over action
- Subtle shifts rather than events
- Awareness without drama

Possible thematic territory:
- Seasons
- Growing
- Small changes
- New routines

Rule:
- Do not reference, imply, or anticipate events, changes, or outcomes from this set prior to Book 13.

---

SET FOUR (Books 19–24)
Title: The Way We Belong

Status:
- Reserved for later canon

Thematic Intent (High-Level Only):
- Belonging as feeling, not identity
- Fit rather than definition
- Connection through shared presence

Possible thematic territory:
- Neighbors
- Visitors
- Community
- Shared quiet moments

Rule:
- This set represents deepening, not conclusion.
- Do not frame earlier books as leading toward an ending.
