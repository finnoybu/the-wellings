# CANON_SETS.md

This document defines the canonical structure of *The Wellings* book franchise.

Sets establish thematic arcs across groups of books.
They do not prescribe plot, pacing, or events within individual stories.

---

## THE WELLINGS (FRANCHISE)

*The Wellings* is a quiet, observational children’s book franchise centered on:
- family
- return
- time
- noticing
- belonging

Stories favor emotional continuity over dramatic action.
Meaning is cumulative and emerges through repetition, tone, and small moments.

---

## CANON SETS OVERVIEW

The franchise is organized into **four canonical sets**, each containing **six books**.

Each set represents a thematic arc rather than a narrative sequence.
Books may be read independently, but sets deepen meaning when read together.

---

## SET ONE (BOOKS 1–6)

### **The Days We Come Home**

This set establishes the emotional and tonal foundation of the franchise.

**Core themes:**
- returning
- daily rhythms
- warmth
- waiting
- quiet reassurance

Stories often center on:
- busy days
- questions without urgency
- small comforts
- the feeling of arriving back where one belongs

The title reads as a memory rather than a chapter,
inviting a sense of familiarity rather than progress.

Canonical series label:

**The Wellings — The Days We Come Home**

---

## SET TWO (BOOKS 7–12)

### **The Time Things Take**

This set mirrors Set One while shifting focus from place to time.

**Core themes:**
- patience
- waiting
- settling
- time passing gently
- acceptance of duration

Stories may explore:
- lost or unstructured time
- pauses
- gradual understanding
- moments that cannot be rushed

The tone grows slightly more reflective while remaining accessible.

Canonical series label:

**The Wellings — The Time Things Take**

Together, Set One and Set Two form a natural pairing:

- Place → then Time  
- Return → then Rest  

---

## SET THREE (BOOKS 13–18)

### **The Changes We Notice**

This set centers awareness rather than action.

**Core themes:**
- small changes
- observation
- growth
- shifting routines
- seasons and transitions

Stories focus on noticing:
- subtle differences
- new patterns
- quiet adjustments

The emphasis is on perception, not resolution.
Nothing dramatic needs to happen for something to feel different.

Canonical series label:

**The Wellings — The Changes We Notice**

---

## SET FOUR (BOOKS 19–24)

### **The Way We Belong**

This set deepens emotional connection rather than concluding the franchise.

**Core themes:**
- belonging
- shared quiet
- neighbors and visitors
- emotional fit

Stories may include:
- expanded community
- gentle social moments
- reassurance of place within the world

Belonging here is emotional, not social.
It reflects comfort and alignment rather than identity or crowd.

Canonical series label:

**The Wellings — The Way We Belong**

---

## NOTES ON SET STRUCTURE

- Sets define *thematic continuity*, not plot progression
- No set represents a beginning or ending
- Emotional depth increases, but tone remains gentle
- Readers may enter the franchise at any set

Sets are intended to grow with the reader
without ever outgrowing the reader.
